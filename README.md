# 🛒 Flipkart Clone

A responsive **Flipkart homepage clone** built using **HTML** and **CSS**. This project recreates the look and feel of the Flipkart landing page and is intended for learning frontend web development and responsive design.

## 🚀 Live Demo

🔗 https://milapbhut.github.io/Flipcakad_clone/

## 📌 Features

- Responsive navigation bar
- Search bar with icon
- Product categories section
- Banner/hero slider
- Product cards
- Offers and deals section
- Footer similar to Flipkart
- Mobile-friendly layout

## 🛠️ Built With

- HTML5
- CSS3

## 📂 Project Structure

```
Flipcakad_clone/

├── assets/             # Images and icons
```

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/milapbhut/Flipcakad_clone.git
```

2. Open the project folder

```bash
cd Flipcakad_clone
```

3. Run the project

Simply open `index.html` in your browser.
